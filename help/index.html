用GitHub免费做网站

一、注册(已有账号跳过)

1、进入→

GitHub

←官网(英语不好的朋友请用百度翻译)

2、点击右上角的“Sign up”(注册)，输入邮箱按提示操作即可(里边提到用钱的没事别乱点，要$0的就行，其它随便选)

二、登录

1、在第一步的第一点基础上点击右上角的三横，再点击下边的“Sign in”即进入登录界面

2、第一个框输邮箱，第二个框输密码，再点绿色按钮的“Sign in”登录（第一次在某设备登录需进入邮箱点击新邮件查看验证码(6位验证码在“6-bit code”后)）

三、建仓库

1、登录状态下，进入→

GitHub仓库新建←

2、第一个框是输仓库名的，仓库名格式：你的github名+“.github.io”

3、勾选倒数第三个选项，即自带一个自述文件，这样才能拥有一个fork分支，其他不要乱改

4、点击绿色按钮新建储存库

四、修改仓库设置

1、登录状态下进入仓库

2、把索引拖到最右边，看到一个Setting（设置）项，点击

3、找到Pages（页面）项，点击

4、把None改为main，然后点save保存设置，这时弹出一个链接，记住它，这将是你的网页的网址

五、上传文件

1、登录状态下进入仓库

2、手机：点击索引下三个点；电脑：点击索引下的“Add a file”（上传一个文件）

3、点击中间的create new file(新建新文件)或下边的upload files(上传文件)

4、文件名自定，内容自定(如果文件名后缀为“.html”则为网页，其它后缀为下载)，输入描述（可以像我一样只写个“der”），点击commit change(提交改变)即可新建/上传(如果你想编辑它，可以覆盖上传或是进入文件，点右上三个点，“Edit it”(编辑它)即可编辑)

六、网页

你的网页/文件在之前出现的 链接+文件夹名(没有则不写)+文件名 里(比如这个页面在https://zhs141.github.io/help/index.html里)
