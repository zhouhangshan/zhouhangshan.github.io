新库：
git clone https://......
cd进新文件夹

添：
git pull origin master        # 将GitHub上的代码当下来合并代码，防止提交新代码的时候起冲突
git add .                     # 将代码添加到仓库
git commit -m "描述"          # 将代码提交到仓库
git push -u origin master     # 命令将代码提交到GitHub

删：
git --help                     # 帮助命令
git pull origin master         # 将远程仓库里面的项目拉下来
dir                            # 查看有哪些文件夹
git rm -r --cached target      # 删除target文件夹
git commit -m '删除了xxx文件'   # 提交,添加操作说明
git push -u origin master      # 将本次更改更新到github项目上去
