运算_比较运算

定义

比较两个值大小关系的运算。
比较运算符的结果为True或False。
Python中的比较运算符与数学中的比较运算相比，运算规则相同；但其中，等于、不等于、大于等于和小于等于的写法不同。

Python和数学中比较运算符对照表
  数学比较运算符 Python比较运算符
大于 > >
小于 < <
等于 = =
不等于 ≠ !=
大于等于 ≥ >=
小于等于 ≤ <=

使用

比较运算可以作为条件语句、循环语句的条件使用。
代码示例：
if 3 > 2:
print('上边那个表格的代码有几十行，累死我了')
运行结果：
上边那个表格的代码有几十行，累死我了
代码示例2：
i = 2
while i == 2:
print('i是2')
运行结果：
i是2
