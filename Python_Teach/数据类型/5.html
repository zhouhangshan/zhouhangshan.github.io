数据类型_字典

定义

字典用于储存数据，是一种可以储存任意数据的数据类型。所有元素写在{}中，每一个元素以一对键和值的形式存在(简称键对值)，之间用逗号隔开。
字典中可以储存1个或多个元素，甚至可以没有元素，没有元素的字典一般叫做空字典。


键对值

字典中的元素由键和值组成，键和值之间用:分隔。
字典中的键通常是字符串、数字，而值可以是任何类型的数据。
字典中的键不可重复，若键相同而值不同，则会保留最后添加的值(同变量)；值可以重复


取值

我们可以通过 字典名[键] 的方式，获取字典中该键对应的值。代码示例：
a = {1:'123',2:999}
print(a[1])123


修改

我们可以通过 字典名[键] = 值 的方式，修改列表中的元素。
代码示例：
a = {1:'张',2:'三'}
a[1] = '李'
print(a)
运行结果：
{1:'李',2:'三'}


添加

我们一般用 字典名[键] = 值 来向字典的末尾添加元素。
代码示例：
a = {1:'张',2:'三'}
a[3] = '喜欢编程魔方和乒乓球'
print(a)
运行结果：
{1:'张',2:'三',3:'喜欢编程魔方和乒乓球'}


遍历

遍历就是按顺序依次访问每个元素。固定句式如下：
for 变量 in 字典:
语句
代码示例：
a = {1:'张',2:'三'}
for a123 in a:
print(a)
运行结果：
1
张
2
三


换行

我们在制作字典时，可以恰当地在逗号后换行。
代码示例：
a = {
1:'张',2:'三',
3:'大聪明'
}
