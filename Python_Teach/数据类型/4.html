数据类型_列表

定义

列表是一种可以储存任意数据的数据类型。储存在列表中的数据称为元素，所有元素写在[]中，元素与元素之间用逗号隔开。
列表中可以储存1个或多个元素，甚至可以没有元素，没有元素的列表一般叫做空列表。


索引

列表中的元素是有顺序的，每个元素都有自己的序号，这个序号叫做索引，从左往右数，从0开始。
代码示例：
a = ['张','三']
# 0 1


取值

我们可以通过 列表名[索引] 的方式，获取列表中该索引位置的元素。代码示例：
a = ['123',999]
print(a[1])999


修改

我们可以通过 列表名[索引] = 值 的方式，修改列表中的元素。
代码示例：
a = ['张','三']
a[1] = 李
print(a)
运行结果：
['李','三']


添加

我们一般用 列表名.append(元素) 来向列表的末尾添加元素。
代码示例：
a = ['张','三']
a.append('喜欢编程魔方和乒乓球')
print(a)
运行结果：
['张','三','喜欢编程魔方和乒乓球']


删除

我们用 列表名.remove(元素) 来删除列表中的元素。
代码示例：
a = ['张','三']
a.remove('三')
print(a)
运行结果：['张']]


清空

我们可以使用 列表名.clear() 来清空列表。
代码示例：
a = ['张','三']
a.clear()
运行结果：
[]


遍历

遍历就是按顺序依次访问每个元素。固定句式如下：
for 变量 in 列表:
语句
代码示例：
a = ['张','三']
for a123 in a:
print(a)
运行结果：
张
三


换行

我们在制作列表时，可以恰当地在逗号后换行。
代码示例：
a = ['张','三',
'大聪明'
]
